## 目录

boltcep+vue+tailwindcss3: 使用vue+tw3开发, 兼容2020版本, 过低版本没测试过, 其中2020开发的时候建议使用22进行, 然后打包可以正常显示, 否则开发的时候会报vite错误, 具体可以看下方boltcep文档

## 工具介绍

[bolt-cep](https://github.com/hyperbrew/bolt-cep) 是一个基于 Vite + TypeScript + Sass 构建的、用于使用 Svelte、React 或 Vue 构建 Adobe CEP 扩展的极速框架
